# JAIN SLEE Skeletons

This folder contains initial Service Building Block skeletons for TrustCall.

## SBBs

- ReputationSbb
- FraudSbb
- WangiriSbb
- SimSwapSbb
- UssdSbb
- ReportingSbb

## Purpose

These classes represent the planned JAIN SLEE service modules. At this stage, they are skeletons used to show the telecom service architecture before deployment to a real RestComm JAIN SLEE container.

## Current Status

- SBB skeleton files created
- Event handler method names defined
- Console output used for validation
- Full RestComm deployment not yet implemented
